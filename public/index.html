URL Shortener Microservice

Example Usage

  1. You pass a Url as parameter and you will recieve a shortened Url in json format
  2. When you visit that shortened Url, it will take you to the original Url link
Example Input:
https://amazing-afternoon.glitch.me/new/https://www.google.com
https://amazing-afternoon.glitch.me/new/http://foo.com:80
Example Output:
{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }
